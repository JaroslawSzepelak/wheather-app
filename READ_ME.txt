1. Aby aplikacja wczytała listę miast w aplikacji należy zainstalować pakiet vue-serwer za pomocą komendy npm install json-server -g a następnie w konsoli przejść
 do katalogu data aplikacji i wywołać komendę json-server city.list.json.
2. Aplikacja uruchamia się po wywołaniu komendy npm run serve w katalogu głównym.
3. Wersja Node.js z jakiej korzystałem: v16.15.0
